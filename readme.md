# mac-dotfiles
Shell configurations for macOS

## commandline tools
### required
- [zplug](https://github.com/zplug/zplug)
- [fzy](https://github.com/jhawthorn/fzy)
  - to support `enhancd`

### optional
- [custom-commands](https://github.com/tomon9086/custom-commands)
- [gibo](https://github.com/simonwhitaker/gibo)
